班级点名器 - 打包说明
=====================

一、打包步骤
-----------

1. 运行 build.bat 批处理文件，它会自动安装所需的依赖并执行打包命令
2. 打包完成后，可执行文件会存放在 dist/ClassRollCall 目录下
3. 将整个 dist/ClassRollCall 目录复制到目标电脑上即可使用

二、使用方法
-----------

1. 双击 dist/ClassRollCall 目录中的 "班级点名器.exe" 文件启动应用
2. 应用会自动打开浏览器，显示点名器界面
3. 如果浏览器没有自动打开，可以手动访问 http://127.0.0.1:5000

三、注意事项
-----------

1. 第一次使用时，请先添加学生名单
2. 学生名单保存在 students.json 文件中，会自动创建
3. 音效文件存放在 static 目录下，确保它们存在且未损坏
4. 应用运行时会在后台启动一个本地服务器，关闭应用时请退出命令行窗口

四、常见问题
-----------

1. 如果启动应用时提示端口被占用，请关闭可能占用 5000 端口的其他应用
2. 如果打包过程中出现错误，请确保已安装 Python 3.6 或更高版本
3. 如果音效无法播放，请检查浏览器设置，确保允许网页播放声音
4. 如果打包时出现中文乱码，确保 build.bat 文件中包含 "chcp 65001" 命令以切换到UTF-8编码

五、技术支持
-----------

如有问题，请参考 README.md 文件或联系技术支持。
